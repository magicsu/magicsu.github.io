Linux常用命令学习记录

发表于 2018-06-23

搜索

-locate: 文件搜索，本地数据库索引搜索

-find: 文件搜索

- find [搜索范围] [搜索条件]
- 支持通配符
    - *: 匹配任意内容
    - []: 匹配中括号内任意一个字符
    - ?: 匹配任意一个字符
- 搜索条件
    - name: 名称
    - iname: 不区分大小写
    - user: 按照所有者搜索
    - atime: 文件访问时间
    - ctime: 改变文件属性
    - mtime: 文件修改内容
        - eg: 查找10天前修改的文件 -> find /var/log/ -mtime +10
        - params: -10 -> 十天内修改的文件, 10 -> 第10天当天修改的文件，+10 -> 10天前修改的文件
    - size
        - eg: 查找大于10k的文件 -> find /var/log/ -size +10k，查找大约20k且小于50k的文件 -> find /var/log/ -size +20k -a -size -50k
        - eg: 查找/etc目录下，大于10M的文件，并显示详细信息 -> find /etc -size +10M -exec ls -lh {} \; (注意分号)
        - eg: 查找/etc目录下，大于10M的文件，并删除 -> find /etc -size +10M -exec rm -rf {} \;
- eg: find / -name xxxx.log

-grep：字符串搜索，使用正则进行匹配

- grep [选项] 字符串 文件名
- cat/tail [选项] 文件名 | grep 字符串    
- eg: 查找包含"hello"的字符串： grep "hello" xx.log

帮助命令

-man 

- man 命令
- eg: man ls

-info

- info 命令
- eg: info ls

-help

- help 命令(shell内部命令)
    - 如何判断命令是否为内部命令，是否存在/bin/可执行命令，若无，为内部命令： whereis 命令
    - eg: whereis cd
- eg: help cd

-help

解压缩

-参数
-c 打包
-x 解压
-v 压缩过程
-f 指定打包后的文件名

-压缩（直接压缩文件夹及其文件）
-gz格式： tar -zcvf xxxx.tar.gz source
-bz2格式：tar -jcvf xxxx.tar.gz2 source

-解压
-gz格式： tar -zxvf xxxx.tar.gz
-bz2格式：tar -jxvf xxxx.tar.bz2
-解压到指定目录： tar -zxvf xxxx.tar.gz -C /xx/xxx

权限

-权限结构

- -rw-r--r-- -> 1:3:3:3 -> 文件类型:u所有者:g所属组:o其他人
- 常用文件类型（3）：- -> 普通文件 / d -> 目录 / | -> 软链接文件

-chmod

- chmod [选项] 模式 文件名
- 选项： -R -> 递归
- 模式1： [ugoa][+-=][rwx]
- 模式2： [mode=421]
- 权限的数字表示： r -> 4   w -> 2   x -> 1
- eg： rwxr-xr-x -> 755 -> chmod 755 文件

-chown
-chown 用户名 文件名

- eg: chown

-权限对文件的作用
-r 可读取文件内容 -> cat/more/head/tail
-w 编辑、新增、修改文件内容（不包括删除文件） -> vi/echo
-x 可执行（最高权限）

-权限对目录的作用
-r 可查询目录的文件名 -> ls
-w 可修改目录结构(最高权限)，如新建/删除/重命名文件、目录 -> touch/rm/mv/cp
-x 可进入目录 -> cd

Shell

-语法类型（主要两种, 可调用echo $SHELL查看）

- Bourne: sh、ksh、Bash、psh
- C: csh、tcsh

-脚本创建后执行方式

- 赋予执行权限，直接运行
    - chmod 755 xx.sh
    - ./xx.sh
- 通过Bash调用执行脚本
    - bash xx.sh

-Bash的基本功能

- 命令别名、快捷键
    - alias: 查看当前列表列表
    - eg: alias ll='ls -l --color=auto'
    - 永久生效：写入环境变量配置文件 -> vi ~/.bashrc
    - 删除别名：unalias 别名
